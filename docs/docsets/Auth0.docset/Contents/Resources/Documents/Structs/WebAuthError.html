Represents an error during a Web Authentication operation.

    The correct method for Token Endpoint Authentication Method is not set (it should be ‘None’). You need to enable PKCE support in your Auth0 application’s settings page, by setting the ‘Application Type’ to ‘Native’ and the ‘Token Endpoint Authentication Method’ to ‘None’. This error does not include a cause.

    Declaration

    public static let pkceNotAllowed: WebAuthError
