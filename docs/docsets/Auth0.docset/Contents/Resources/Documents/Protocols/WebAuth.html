    Starts the WebAuth flow.

    Auth0
    .webAuth(clientId: clientId, domain: "samples.auth0.com")
    .start { result in
        switch result {
        case .success(let credentials):
            print("Obtained credentials: \(credentials)")
        case .failure(let error):
            print("Failed with \(error)")
    }
}

    Any on going WebAuth Auth session will be automatically cancelled when starting a new one, and it’s corresponding callback with be called with a failure result of WebAuthError.userCancelled.

    Starts the WebAuth flow.

    Auth0

    .publisher()

        if case .failure(let error) = completion {
            print("Failed with \(error)")
        }
    }, receiveValue: { credentials in
        print("Obtained credentials: \(credentials)")
    })
